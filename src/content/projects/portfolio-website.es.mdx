# Sitio Web Portfolio

Este es mi sitio web portfolio personal construido con tecnologías web modernas. El sitio presenta un diseño limpio y responsivo, y muestra mis proyectos y habilidades como desarrollador full-stack.

## Características

- **Diseño Responsivo**: Funciona perfectamente en todos los dispositivos
- **Modo Oscuro**: Alternar entre temas claro y oscuro
- **Internacionalización**: Disponible en inglés y español
- **Rendimiento Rápido**: Construido con Astro para velocidades de carga óptimas
- **Optimizado para SEO**: Meta tags apropiados y HTML semántico

## Tecnologías Utilizadas

- **Astro**: Para generación de sitios estáticos y rendimiento óptimo
- **TypeScript**: Para seguridad de tipos y mejor experiencia de desarrollo
- **Tailwind CSS**: Para estilos basados en utilidades
- **Alpine.js**: Para interactividad ligera
- **MDX**: Para gestión de contenido

## Desafíos y Soluciones

Uno de los principales desafíos fue implementar un sistema de internacionalización robusto que funcione perfectamente con la generación estática de Astro. Lo resolví creando rutas dinámicas y una utilidad de traducción que filtra el contenido según el idioma actual.
